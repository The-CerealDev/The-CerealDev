 # David

I'm a Sixth Form Student, interested in maths and programming.

### Current Projects

*   **[speed-maths](https://github.com/The-CerealDev/speed-maths)** — An open-source community building daily TMUA, SMC, and BMO1 practice sheets. To make sure we don't ship bad maths, I created a Python/LaTeX pipeline that computationally verifies every submitted answer rather than relying on LLM hallucination. If you like maths, come contribute.
*   **[py-scrape](https://github.com/The-CerealDev/py-scrape)** — A multithreaded Selenium scraper I wrote to pull regulatory filings from the London Stock Exchange. Uses PyShadow to pierce Shadow DOMs and automates the data validation so I don't have to do it by hand.
*   **[fode-tools](https://github.com/The-CerealDev/fode-tools)** — Python implementations of numerical methods for first-order differential equations (Euler's & Midpoint). Mostly wrote this to check my own A-Level Further Maths work.
*   **[task-tamer](https://github.com/SFH6-Devs/task-tamer)** — A shared codebase for the SFH6 Dev Club I run. I use this to teach Year 7–11 students how to use Git, open pull requests, and work as a team without breaking production.
